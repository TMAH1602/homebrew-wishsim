# Homebrew tap for WishSim

Install [WishSim](https://github.com/TMAH1602/WishSim) on macOS with:

```sh
brew tap TMAH1602/wishsim
brew install wishsim
```

Upgrade later releases with:

```sh
brew update
brew upgrade wishsim
```

Run `wishsim` directly in Kitty for full-resolution character artwork. Terminal multiplexers such as Zellij may block Kitty's graphics protocol; the portable terminal-art fallback remains available elsewhere.
